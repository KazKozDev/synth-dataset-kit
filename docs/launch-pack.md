# Launch Pack

Use this checklist when turning the repository into a public product launch.

## Required Assets

- one benchmark-proof page
- one published Hugging Face dataset
- one proof bundle
- one short terminal demo
- one concise README happy path

## Distribution Assets

- PyPI package page
- Hugging Face dataset page
- GitHub README hero section
- one post for r/LocalLLaMA
- one post for Hacker News

## Messaging

- input: `20-50` real examples
- output: usable, auditable fine-tuning dataset
- differentiator: faster than framework-heavy synthetic-data stacks

## Minimum Public Launch Set

1. customer support showcase
2. one non-support showcase
3. one benchmark-proof run
4. one quality report screenshot
5. one Hugging Face URL
