# Distribution Plan

This file exists so distribution is treated as product work, not “later”.

## Primary Distribution Surface

- Hugging Face datasets

## Secondary Distribution Surfaces

- PyPI
- GitHub README
- r/LocalLLaMA
- Hacker News
- X / LinkedIn threads with proof artifacts

## Minimum Launch Pack

- one public Hugging Face dataset
- one public proof bundle
- one benchmark-proof page
- one concise README happy path
- one terminal demo GIF or transcript

## Post Templates To Produce

### Launch Post

- problem: frameworks are too heavy
- input: 20-50 real examples
- output: auditable dataset + report + HF bundle
- proof: timings + quality gate + contamination result

### Case Study Post

- seed count
- generated count
- average quality
- contamination hits
- publish URL
- uplift or next-step fine-tune result

## What Counts As Success

- 3 external users run the tool
- 1 inbound request from a dataset/model page
- 1 repeat user with a second seed set
